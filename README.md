# PrezBigCorpoFood

## Description du projet
Ce projet est une présentation collaborative en HTML réalisée pour l’entreprise fictive Big_Corpo_Food.

Chaque membre de l’équipe contribue en créant une diapositive au format HTML.

## Technologies utilisées
- HTML
- Git
- GitHub

## Travail en équipe
Chaque membre doit :
- Créer son propre fichier HTML (ex : 1.html, 2.html, 3.html…)
- Ajouter son fichier avec `git add`
- Faire un commit
- Push sur la branche main

Avant de commencer à travailler, toujours exécuter :

git pull origin main

## Objectif
Apprendre à collaborer sur Git et GitHub dans un environnement de travail en équipe.
